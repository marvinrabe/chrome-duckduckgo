h1. DuckDuckFallBack

Chrome extension to add a Google search button to the DuckDuckGo results.

Extends the DuckDuckGo search results with a button to switch easily to the Google search.

Available on the "Chrome Webstore":https://chrome.google.com/webstore/detail/duckduckfallback/bgghgnejijjooigfhkfmhophopjpmfpe .
